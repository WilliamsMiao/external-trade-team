# 项目目标

## 使命
建立完全由 Agent 自主工作的外贸团队，自动化处理订单、客户、供应链、财务等业务流程。

## 阶段目标

### Phase 1: 基础设施
- [x] Docker 环境搭建
- [x] 数据库设计
- [ ] MiniMax API 集成
- [ ] Agent 框架搭建

### Phase 2: 核心功能
- [ ] 客户管理 Agent
- [ ] 订单处理流程
- [ ] 供应链协同
- [ ] 财务审批流

### Phase 3: 智能化
- [ ] 多 Agent 协作
- [ ] 外部系统集成 (CRM, 物流, 支付)
- [ ] 智能决策引擎
